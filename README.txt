README - AcademeForge static site (clone)
Files:
- index.html : main page (references images hosted on insayy.com)
- styles.css : styling
- script.js : small interactive behaviors

Notes:
1. Images are currently hotlinked from insayy.com. To make the site fully offline:
   - Download the images (hero.png, asset.png, startupindia.png, startupbihar.png) and place them in an 'images' folder.
   - Update the <img src="..."> paths in index.html to point to local files (e.g. images/hero.png).

2. Branding:
   - Replace text "AcademeForge" and email/team links with your real brand info.
   - Replace pricing/claims as needed.

3. This is a simple static recreation for AcademeForge inspired by the layout and copy of insayy.com.
   - If you want a pixel-perfect replica (fonts, animations, original JS), I can extract more assets or convert to a Next.js project.

How to use:
- Unzip and open index.html in your browser.
